drill Function

Defined in: index.js

Drill into a component and start kicking. This is the main export of the library.

Example

var drill = require('react-drill');
var MyComponent = require('./MyComponent');
var component = ReactDOM.render(MyComponent, document.createElement('div'));

drill(component)
  .find('button')
    .click()
;
