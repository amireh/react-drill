RCEScope Class, Scope

A drilling scope for manipulating Draft.js Rich Content Editor components.

You don't need to instantiate this directly, use warpToRCE instead.

Constructor

RCEScope

drill(subject)
  .warpToRCE()
    .setContent('Hello World!')
    .select('World')

// assuming there's a button that will apply a BOLD inline style to the
// editor's current selection:
drill(subject)
  .find('button', m.hasText('Toggle BOLD'))
    .click()

drill(component)
  .warpToRCE()
    .assertHasInlineStyle('BOLD', { text: 'World' })

    getCaretPosition()

    Retrieve the character offset of the current selection from the beginning of the editor content.

      Examples

      "Hello World!"
       ^
 0     6

drill(subject).warpToRCE().getCaretPosition() === 6;

    getRawState() -> Object

    Retrieve the "raw state" of the editor's current content.

    getSelectedText() -> String

    Get the currently selected text as reported by the editor's state.

    getSelection() -> DraftJS.SelectionState

    Get the current selection as reported by the editor's state.

    insertLineBreak()

    Insert a line break through a RETURN key-press.

    insertSoftLineBreak()

    Insert a "soft" line break through a RETURN plus SHIFT key-press.

    keyDown(keyCode: Number, options: Object)

    A version of react-drill's Actions.keyDown that:

    1. supports ctrl/meta/shift modifiers
    2. operates on the editor's root node
    3. does not focus the node, as it is not necessary here

    setCaretPosition(index: Number)

    Move the caret to the specified character offset. This generates a collapsed selection.

    setContent(content: String, options: Object?) -> RCEScope

    Adjust the text contents of the RCE.

    Parameters (3)

    content
    		String
    options
    		Object?
    options.interpretNewlines
    		Boolean?

    Turn this on if you have line-breaks in the content (\n) that you want to turn into RCE breaks (resulting into <p /> nodes).

    This is a convenience option that is equivalent to emitting keyDown events with the 'Enter' (KC 13) key.
