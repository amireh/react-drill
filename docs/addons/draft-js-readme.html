Draft.js drill addon

This addon provides a custom scope for interacting with the Draft.js rich-content editor.

The scope is defined in RCEScope and can be accessed by warping to the editor component as shown below.

import { drill } from 'react-drill'
import 'react-drill/addons/draft-js'

drill(component)
  .warpToRCE()  // => RCEScope

Dependencies

  • chai
  • draft.js v0.10
  • react-dom v15
